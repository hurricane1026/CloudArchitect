# Appendix A: Cloud Provider Comparison

## Feature Comparison Matrix

| Feature | AWS | Azure | GCP | Alibaba Cloud |
|---------|-----|-------|-----|---------------|
| **Compute** | EC2 | Virtual Machines | Compute Engine | ECS |
| **Containers** | ECS, EKS, Fargate | AKS, ACI | GKE, Cloud Run | ACK |
| **Serverless** | Lambda | Functions | Cloud Functions | Function Compute |
| **Object Storage** | S3 | Blob Storage | Cloud Storage | OSS |
| **Block Storage** | EBS | Managed Disks | Persistent Disk | Cloud Disk |
| **Relational DB** | RDS, Aurora | SQL Database | Cloud SQL | RDS |
| **NoSQL DB** | DynamoDB | Cosmos DB | Firestore, Bigtable | Table Store |
| **CDN** | CloudFront | Azure CDN | Cloud CDN | Alibaba CDN |
| **Load Balancer** | ELB, ALB, NLB | Load Balancer | Cloud Load Balancing | SLB |
| **DNS** | Route 53 | Azure DNS | Cloud DNS | Alibaba Cloud DNS |
| **VPN** | VPN Gateway | VPN Gateway | Cloud VPN | VPN Gateway |
