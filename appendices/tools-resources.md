# Appendix B: Useful Tools and Resources

## Infrastructure as Code

- **Terraform**: Multi-cloud infrastructure provisioning
- **Pulumi**: Modern infrastructure as code using programming languages
- **AWS CloudFormation**: AWS native IaC
- **Azure Resource Manager**: Azure native IaC
- **Google Cloud Deployment Manager**: GCP native IaC

## CI/CD Tools

- **Jenkins**: Open-source automation server
- **GitHub Actions**: CI/CD integrated with GitHub
- **GitLab CI/CD**: Integrated CI/CD platform
- **CircleCI**: Cloud-based CI/CD
- **Travis CI**: Continuous integration service
- **AWS CodePipeline**: AWS native CI/CD
- **Azure DevOps**: Microsoft's DevOps platform

## Monitoring and Observability

- **Prometheus**: Open-source monitoring system
- **Grafana**: Analytics and monitoring platform
- **Datadog**: Cloud monitoring service
- **New Relic**: Application performance monitoring
- **Elastic Stack (ELK)**: Elasticsearch, Logstash, Kibana
- **Splunk**: Data analytics platform
- **AWS CloudWatch**: AWS native monitoring
- **Azure Monitor**: Azure native monitoring
- **Google Cloud Monitoring**: GCP native monitoring

## Container Tools

- **Docker**: Container platform
- **Kubernetes**: Container orchestration
- **Helm**: Kubernetes package manager
- **Istio**: Service mesh
- **Linkerd**: Lightweight service mesh
- **Rancher**: Kubernetes management platform

## Security Tools

- **HashiCorp Vault**: Secrets management
- **AWS Secrets Manager**: AWS secrets management
- **Azure Key Vault**: Azure secrets management
- **Snyk**: Security scanning for code and containers
- **Aqua Security**: Container security
- **Twistlock/Prisma Cloud**: Cloud native security

## Cost Management

- **Kubecost**: Kubernetes cost monitoring
- **CloudHealth**: Multi-cloud cost management
- **Spot.io**: Cloud cost optimization
- **AWS Cost Explorer**: AWS cost analysis
- **Azure Cost Management**: Azure cost analysis
